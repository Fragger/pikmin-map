Pikmin Bloom Map

This map helps you find the likely locations of decorations in the game Pikmin Bloom, sourced from Open Street Map data.

Disclaimer: This application is not affiliated with Nintendo or Niantic Lab, and is currently developed and maintained by pixelpirate.

How does it work?

The game collects landscape, road, and building data from external databases. It primarily comes from Open Street Map (OSM) and occasionally Foursquare.

This map utilizes OSM data exclusively. It retrieves objects with the amenity=restaurant tag to display restaurant decorations.

OSM lacks a direct tag-based retrieval function, so this application employs the Overpass-Turbo tool.
